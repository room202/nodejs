## インストールの確認

コマンドプロンプトを起動して、下記コマンドを実行して  
バージョンが表示されたらインストール成功

### Node.jsのインストール確認

```bash
node --version
```

![](images/013.png)

### npmコマンドの確認

```bash
npm --version
```

![](images/014.png)

### npxコマンドの確認

```bash
npx --version
```

![](images/015.png)
